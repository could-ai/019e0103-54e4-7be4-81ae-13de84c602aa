# Kids Story Posting Calendar

A 30-day social media posting calendar designed specifically for a kids' story account. The app features daily concepts, content formats (Carousels, Reels, Stories), optimal posting times based on audience behavior, engaging hooks, and strategic calls to action (CTAs).

## Features
- **30-Day Grid View**: See the whole month at a glance with color-coded content formats.
- **Responsive Layout**: Works beautifully on mobile, tablet, and desktop.
- **Daily Details**: Tap any day to see the full concept, hook, CTA, format, and optimal posting time.
- **Optimal Timings**: Strategically placed posting times to hit peak audience engagement for parents and caregivers.

## Tech Stack
- Flutter
- Material 3 Design

## Setup / Run
1. Ensure Flutter is installed.
2. Run `flutter pub get`.
3. Run `flutter run` to launch on your connected device or web browser.
